Bootstrap


Twitter Bootstrap

Bootstrap is the most populat HTML, CSS and JS framework for developing responsive, mobile first projects on the web.
  • Free front-end framework
  • Templates for typography, forms, buttons, tables, navigation, ...
  • Responsive design: automatical adjustments for all screen sizes

Usage


<!DOCTYPE html>
<html lang="en">
<head>
 <title>Bootstrap Skeleton</title>
 <meta charset="utf-8">
 <meta name="viewport" content="width=device-width, initial-scale=1">
 <!-- Include Bootstrap css, jQuery js and Bootstrap js from CDN -->
 <link rel="stylesheet"
href="http://maxcdn.bootstrapcdn.com/bootstrap/3.2.0/css/bootstrap.min.css">
 <script src="https://ajax.googleapis.com/ajax/libs/jquery/1.11.1/jquery.min.js"></script>
 <script src="http://maxcdn.bootstrapcdn.com/bootstrap/3.2.0/js/bootstrap.min.js"></script>
</head>
<body>
<!-- Content -->
</body>
</html>

Grid System

  • Fluid grid system scales up to 12 columns
  • Columns will re-arrange depending on the screen size
  • The total columns width of a row should sum up to 12 in total
  • Add rows 
    <div class="row">
  • Add columns 
    <div class="col-[device]-[width]">
    • [device] is xs|sm|md|lg
    • [width] is a number between 0 and 12

jQuery Mobile


jQuery Mobile

jQuery Mobile is a HTML5-based user interface system designed to make responsive web sites and apps that are accessible on all smartphone, tablet and desktop devices.
―jquerymobile.com

jQuery Mobile provides

  • Navigation / Pages (reduced scripting necessary)
  • Widgets (Toolbar, Popup, Listview, ...)
  • CSS Framework (Buttons, Themes, ...)
  • Cross-Platform (iOS, Android, BlackBerry, Windows, ...)

Use jQuery Mobile

  • Offline: download from www.jquerymobile.com and add in project 
    
<link rel="stylesheet" href="jquery.mobile-1.4.5.min.css" />
<script src="jquery-1.11.1.min.js"></script>
<script src="jquery.mobile-1.4.5.min.js"></script>
  • Or online: use directly from CDN
    • http://jquerymobile.com/download/

Pages & Navigation


<html><body>
	<!-- Start of first page -->
	<div data-role="page" id="foo">
		<div data-role="header">
			<h1>Foo</h1>
	</div><!-- /header -->
	<div role="main" class="ui-content">
		lt;p>I'm the first (shown) page.</p>
		<p><a href="#bar">Go to: bar</a>
		</p>
	</div><!-- /content -->
	<div data-role="footer">
		<h4>Page Footer</h4>
	</div><!-- /footer -->
</div><!-- /page -->
