Introduction

This is the practical task assignment for the Interaction Programming Block Course. The block course is taught at the Institute of Ergonomics of the Technische Universität München in the context of master courses in Human Factors Engineering, Mechanical Engineering, and Computer Sciences in the . This document is authored by Markus Zimmermann, David Schopf, and Niklas Lütteken.

The exercises aim to learn the web technologies HTML5/CSS3/JavaScript by developing a weather application during the course.

The Vision

If you follow the task assignments, you will create a weather app looking similar to the image below:

You can experience the "best practice" weather app and browse its source code online, or clone the full Git repository (including exercises, slides, and weatherApp). We recommend that you use the source code of the sample weather app only for clarification or as a reference for wrap-up and first implement your own app. It doesn't have to look or work identically, we encourage you to implement at your own speed and test different approaches. The tutors will be happy to provide you any assistance and inspiration.

Expectations, requirements, and grading:

Your final grade will consist of a open-internet practical exam.

Bring your own notebooks with your favourite authoring environment. Ensure, that you have internet access (for the assignments, the disposal and the starter pack). The exam will be "open-book" and "open-internet", you may use any references and tools, except file-sharing or messengers! You will be given requirements (ca. 20% HTML, ca. 30% CSS, ca 50% JS), which you have to fulfill. Implementing 50% of the requirements is suficcient for passing the exam (grade 4.0). Environment

HTML editing

  • Recommended Editor: Brackets
  • Recommended Browser: Chrome

Test the live preview of Brackets. In case, you’re experiencing problems, consider the Brackets livedev troubleshooting document.

Versioning control

Create a free account at GitHub.

Download and install the Windows or Mac client.

Create a new repository, add your index.html file, commit and sync your changes. Repeat this process regularly throughout this course, this is how you can save different versions of your progress online (and share them with lecturers, tutors and other students). During the semester, we will intensely use GitHub in the Interaction Prototyping Practical Course.
