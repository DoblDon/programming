D3

Box Model


Box model

Source: w3schools.com

  • Boxes are wrapped around all HTML elements
  • Margin: Cleared area around border (transparent, no background-color)
  • Border: Goes around padding and content (background color)
  • Padding: Cleared area around content (background color)
  • Content: Where text and images appear
CSS Box Model

Box: Block vs. Inline

  • Block
    • Width: maximized
    • Height: as necessary
    • Forces line break
  • Inline
  • Width/Height: as necessary
  • No line break
View example on fiddle

Box: Width & Height

  • Set width and height of content area, e.g. 
width: 250px;
padding: 10px;
border: 5px solid gray;
margin:10px;
  • Size: Height/Width + Padding + Border + Margin

(width) 250px

(left + right padding) + 20px

(left + right border) + 10px

(left + right margin) + 20px

= 300px

Metrics

  • %: relative to element/parent size 
    line-height:120%;
  • cm: absolute size 
    top:2.54cm;
  • em: relative to font height 
    font-size:1.2em;
  • ex: relative to lowercase x 
    font-size:1.3ex;
  • in: absolute size 
    border-width:0.1in;
  • mm absolute size 
    margin-bottom:10mm;
  • pc: absolute size Pica [12pt] 
    line-height:1.2pt;
  • pt: absolute size Point [1/72in] 
    font-size:12pt;
  • px: absolute size Pixel, relative size on different resolutions: 72dpi, 300dpi,.. 
    border-width:3px;
