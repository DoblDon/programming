Meter

<meter value="0.6">
60%</meter>

optimum="5">1 out of

    • high: range that is considered to be high
    • low: range that is considered to be low
    • max: maximum value of the range
    • min: minimum value of the range
    • optimum: what is the optimal value for the gauge

HTML5 support

...features not supported in all browsers (the same way)

Solution

  • html5Forms JavaScript framework, emulating HTML5 functionalities („polyfill“) in unsupported browsers
  • Explanation and Download:
    • Creating Cross Browser HTML5 Forms Now, Using modernizr, webforms2 and html5Forms

html5Forms

Get it started:

  • Download html5forms
  • Extract folder shared to your project folder
  • Embed 2 JS scripts into your <head>:
<script type="text/javascript" src="shared/js/modernizr.com/Modernizr-
2.5.3.forms.js"></script>
<script type="text/javascript"
data-webforms2-support="validation,autofocus,placeholder,range,date,color"
data-lang="en"  
src="shared/js/html5Forms.js" >
</script>

Number (Spinner) / Range (Slider)

<input type="number" min="10" max="100" value="50">
<input type="range" min="0" max="10" step="1" value="5">

  • Number selection (often displayed as spinner
  • Range selection (displayed as slider
  • min/max defines range
  • step defines legal interval values
  • value defines default value

Required

Name*:

View example on fiddle

Disabled / read only

<input type="text" value="do not modify" readonly>
<input type="checkbox" disabled>
<input type="submit" disabled>


  • Fields may be disabled or set readonly
  • Usage
    • readonly elements are (temporary) output elements
    • disabled elements are grayed out
View example on fiddle

Placeholder

<input type="text" name="fname" placeholder="First name">
  • placeholder specifies short hint
    • describes expected value (e.g a sample or short description)
    • hint is displayed in the field when empty
    • hint disappears when the field gets focus
View example on fiddle
