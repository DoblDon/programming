Examples: Sortable IV

  • Define a custom function to resort arrays: 
    function updateArraySorting(oldArray, newPositions) {
// Initialize the new array
var newArray = new Array();
// Go trough new positions and build sorted array
for (var p=0; p<newPositions.length;p++) {
// get current position (element with index p)
id = newPositions[p];
// add the object at this position from the old array to the new array
newArray.push(oldArray[id]);
}
// Return the sorted array
return newArray;
}
  • Array indices [0], [1], [2] match ID attributes
  • We want to update the positions, using the array with the current order of id values
