Variable Scope

  • Local variable declared within a JavaScript function:
    • can only be accessed from within that function (the variable has local scope)
    • variables with the same name in different functions possible (only recognized locally by the function)
    • deleted as soon as the function is completed
  • Global variable declared outside a JavaScript function:
    • all scripts and functions on the web page can access it
    • deleted when the page is closed

Push/Pop

  • Push adds new items to the end of an array 
    var letters = new Array();
						letters.push("C")
    C 
    letters.push("D", "E", "F")
    C D E F
  • Pop removes the last element of an array, and eturns that element 
    var letter = letters.pop();
						alert(letter);  // F

Shift/Unshift

  • Shift removes the first element of an array, and returns that element
  • Unshift adds new element to the beginning of an array

    Combination of push and shift allows queue!

Splice

  • Splice adds/removes items to/from an array, and returns the removed item(s)
  • Signature
    • index: at what position to add/remove items. Negative values: position from the end of the array
    • howmany: number of items to be removed (0: no items will be removed)
    • values: Optional. The new item(s) to be added to the array
var spliceArray = ["do", "re", "fa", "fa", "so"];
spliceArray.splice(2, 0, "mi");
alert(spliceArray); // do, re, mi, fa, fa, so
spliceArray.splice(4, 1);
alert(spliceArray); // do, re, mi, fa, so
