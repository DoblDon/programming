JSON


JSON

  • JavaScript Object Notation
  • syntax for storing and exchanging data
  • language independent
  • quicker to read and write than XML
JSON

JSON Syntax

  • JSON data is written in name/value pairs in double quotes: 
    "firstName":"John"
  • JSON objects are written inside curly braces
    {"firstName":"John", "lastName":"Doe"}
  • JSON Arrays are written inside square braces
    "employees":[
    {"firstName":"John", "lastName":"Doe"}, 
    {"firstName":"Anna", "lastName":"Smith"}, 
    {"firstName":"Peter","lastName":"Jones"}
]
  • Accessing entries 
    employees[0].firstName //John

JavaScript and JSON

  • JSON data can be converted into native JavaScript objects 
    var obj = JSON.parse(text);
  • Vice versa: objects can be converted into JSON text 
    var obj = JSON.parse(text);

Web Storage API

  • Web Storage API provides mechanisms by which browsers can store key/value pairs
  • Two mechanisms:
    • sessionStorage maintains a separate storage area for each given origin that's available for the duration of the page session (as long as the browser is open, including page reloads and restores)
    • localStorage persists even when the browser is closed and reopened

Web Audio API

  • Web Audio API provides mechanisms for mixing, processing and synthesizing audio in web applications
  • AudioContext is used for managing and playing all sounds
  • short- to medium-length sounds can be fetched from an audio buffer via a XMLHttpRequest
  • Supported file formats: WAV, MP3, AAC, OGG

Playing sounds

  • playSound() can be called everytime someone presses a key
  • start(time) schedules sound playback
function playSound(buffer) {
var source = context.createBufferSource(); // creates a sound source
source.buffer = buffer;                    // tell the source which sound to play
source.connect(context.destination);       // connect the source to the context's destination (the speakers)
source.start(0);   // play the source now
}

Google Map Geolocation API

What is Geocoding?

  • Geocoding is the process of converting addresses into geographic coordinates
  • Reverse Geocoding is the process of converting geographic coordinates into a human-readable address
  • Geocoding API requests a JSON response using an URL of the form: 
    https://maps.googleapis.com/maps/api/geocode/json?key=API_KEY&address=ADDRESS
