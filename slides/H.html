JSON


JSON

  • JavaScript Object Notation
  • syntax for storing and exchanging data
  • language independent
  • quicker to read and write than XML
JSON

JSON Syntax

  • JSON data is written in name/value pairs in double quotes: 
    "firstName":"John"
  • JSON objects are written inside curly braces
    {"firstName":"John", "lastName":"Doe"}
  • JSON Arrays are written inside square braces
    "employees":[
    {"firstName":"John", "lastName":"Doe"}, 
    {"firstName":"Anna", "lastName":"Smith"}, 
    {"firstName":"Peter","lastName":"Jones"}
]
  • Accessing entries 
    employees[0].firstName //John

JavaScript & JSON

  • JSON data can be converted into native JavaScript objects 
    var obj = JSON.parse(text);
  • Vice versa: objects can be converted into JSON text 
    var obj = JSON.parse(text);

APIs


Web APIs to share data

  • Application Programming Interface
  • content can be dynamically posted and updated in multiple locations on the web
  • query data via HTTP requests
  • JSON data as response in the background via asynchronous loading (AJAX)

Usage

  • Connectivity:allowing you to communicate with the server
  • Offline & Storage: allowing webpages to store data on the client-side locally and operate offline more efficiently
  • Multimedia: controlling aufio on the web
  • Device access: allowing for the usage of various input and output devices

Retrieving JS objects

  • jQuery AJAX Request 
    $.ajax(url, function(data) {
// DO SOMETHING WITH THE RESULT
});
  • basic response: JSON String
  • Usually parsed as JS object by jQuery 
    {"first":"klaus", "last":"bengler"}
  • Shorthand version to load JSON via AJAX 
    $.getJSON('object.txt', function(data) {
alert("Hallo " + data.first + ' ' + data.last);
});
