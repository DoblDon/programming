	<h1>Heading</h1>

  • Tags are part of the element: and keywords in <>
  • Pairs: opening <> and closing </>
  • Element content is everything between start and end tags
  • Elements can be nested (contain other HTML elements)

Structure

Sectioning & Grouping


<section><h1><p><br>

  • Section defines logical sections (chapters, ...)
  • Headings are important information
    • <h1> most important
    • <h6> least important
  • Paragraphs <p>
  • Line breaks <br>

<h1>My book</h1>
<section>
		<h1>First chapter</h1>
		<p>Content</p>
</section>
<section>
		<h1>Second chapter</h1>
		<p>Content</p>
		<p>Another<br>paragraph</p>
</section>

<article>

  • Independent logical parts are grouped using the article tag
  • Logical part in a forum: e.g. posts (author, content, signature, date, comments, ...)

Textformatting

  • Formatting output (Inline): words, expressions, sentences, etc.

B4

Hyperlinks


Hyperlinks

  • are references for structuring HTML document(s)
  • Click link jump to another document!
    Interaction: load different HTML pages

<a href="destination.html">Link text</a>

Href: Target URL

  • Relative
    • Same Directory 
      
href="menu-playback.html"
    • Subfolder 
      
href=“musicplayer/menus/menu-main.html"
    • Parent Directory 
      
href="../version1.html"
  • External Link 
    
href="www.google.de/index.html"
  • Absolute Link
    DON'T USE: will not work on other PC 
    
href="c:\IPP\start.html"
