Client/Server & HTTP

HTML...

  • Hypertext Markup Language
  • Describes structure of websites(logically)
  • Developed by web prime father Tim Berners-Lee, most widespread file format worldwide

...presented in browsers

...is hypertext

  • Structures text (Elements: headline, paragraph, list, table, ...)
  • Classifies sections and document structure (Semantics: header,content, footer, navigations)
  • Allows links to other pages (hypertext & interaction)
  • Images and multimedia are referenced
  • Provides interfaces for:
    • CSS (Cascading Style Sheets) to style the content
    • JS (JavaScript) to realize the interaction

...markup is hierarchic

  • Document contains header and content
  • Content contains an arbitrary amount of structuring elements (e.g. navigation, textblock)
  • Navigation could contain a list (that contains elements)
  • Text block could contain headings, paragraphs, graphics
  • Word in paragraph could be bold or red,...
  • HTML follows tree structure

...contains hyperlinks

  • Links reference other documents
    • same website
    • somewhere in the word wide web

...is plain text

CSS

  • Cascading Style Sheets
  • Supplement language for HTML:
    • Formatting content
    • Styling website and structure
  • Definecentral style sheets for different classes of content (all headlines, one specific paragraph, ...)
  • Is, like HTML, plain text (can be edited in any other editor

Scripting in JS

  • JavaScript
  • Scripting language 
    var result = 11*3+5;
if (result > 30)
	alert(result + "is larger than 10");
  • Access and manipulate HTML/CSS dynamically 
    document.getElementById('result').innerHTML = result;
  • Executed by the browser (client side)

DOM

  • Document Object Model
  • Browser builds DOM object from the (declarative) HTML
  • CSS (declarative) specifies how DOM nodes are rendered
  • JavaScript (procedural)
    • reeds and modiefies DOM
    • reacts on events and user interaction

Klaus Birkenbühl

JS/DOM example


<!DOCTYPE html>
<html>
<head>
<script type="text/javacript">
	function greet() {	/* procedural */
		time = new Date(); /* variable */
		user = document.getElementById('user').value; /* DOM interaction */
		message = "Hello " + user + ", it is " + time.toLocaleTimeString();
		alert (message); /* function */
	}
</script>
</head>
<body>
	<input type="text" id="user" placeholder="Your Name"> <!-- HTML element with id -->
	<button onclick="greet();">Greet</button> <!-- user interaction -->
	</body>
</html>
